About This Planner

This Student Planner is designed to help students manage assignments, deadlines, and academic workload efficiently. It focuses on simplicity, clarity, and productivity.

Built using HTML, CSS, Bootstrap 5, and JavaScript with a clean MVC architecture, it demonstrates real frontend engineering: filtering, sorting, drag-and-drop reordering, dark mode, PDF export, and persistent localStorage state.
